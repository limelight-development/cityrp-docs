    Module internet

    Shared library utilities to help internet do shit.

    Module Overview

    Client Functions

    INTER_RUNSTR_SEND(inStr) Send an input string to the server.
    INTER_RUNSTR_MAKEDERMA() Create the runstring derma shell.

    Shared Functions

    var_dump(var[, lvl=1]) Equivilent to PHP's Var Dump.
    INTERNET() Function to get Internet's Entity, if he's online.
    INTER_CHAT(...) Output chat to Internet's Chatbox
    INTER_EYE() Get Internet's Eye Trace.
    INTER_ENT() Get Internet's Eye Trace Hit Entity.

    Server Functions

    INTER_RUNSTR(ply) Run a multi-part string as defined in INTERGETPART and INTERNETRUNSTRINGS.
    INTER_GETPART(ply, text, first, last) Recieve and add a multipart string to the runstring table.

    Client Functions

    INTER_RUNSTR_SEND(inStr)
    Send an input string to the server.

    Parameters:

    • inStr string The input string to send to the server.
    INTER_RUNSTR_MAKEDERMA()
    Create the runstring derma shell.

    Shared Functions

    var_dump(var[, lvl=1])
    Equivilent to PHP's Var Dump. Turns a variable into a human readable representation.

    Parameters:

    • var The variable to dump.
    • lvl number The indent level to use. (default 1)

    Returns:

      string The dumped variable.
    INTERNET()
    Function to get Internet's Entity, if he's online.

    Returns:

      Player or boolean Returns the player object if online, false otherwise.
    INTER_CHAT(...)
    Output chat to Internet's Chatbox

    Parameters:

    • ... Any Input to display to internet.
    INTER_EYE()
    Get Internet's Eye Trace.

    Returns:

      table or boolean Returns eye trace table on success, false otherwise.
    INTER_ENT()
    Get Internet's Eye Trace Hit Entity.

    Returns:

      Entity or boolean Returns entity on success, false otherwise.

    Server Functions

    INTER_RUNSTR(ply)
    Run a multi-part string as defined in INTERGETPART and INTERNETRUNSTRINGS.

    Parameters:

    • ply Player The player attempting to run the command. Must have luarun access, else will cause a security alert.
    INTER_GETPART(ply, text, first, last)
    Recieve and add a multipart string to the runstring table.

    Parameters:

    • ply Player The player attempting to run the command. Must have luarun access, else will cause a security alert.
    • text string The part of string to add to the table.
    • first bool If this is the first part of a multipart string (wipe the table)
    • last bool If this is the last part of a multipart string (run it after adding)
